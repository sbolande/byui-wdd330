Transforms

  • Transform lets you "translate, rotate, scale, and/or skew any element on the page."
  • There are both 2D and 3D transformations.
  • Transforms do NOT work on inline elements, add some other display value than inline for transformations.

Translation

  • Translation lets you "move elements left, right, up, or down."
  • The translate(x, y) function moves elements right 'x' and down 'y'.

Scaling

  • The following scaling functions scale an element by given factors:
    • scale(x, y) scales horizontally 'x' and vertically 'y'.
    • scale(n) scales horizontally and vertically 'n'.
    • scaleX(x) scales horizontally 'x'.
    • scaleY(y) scales vertically 'y'.

Rotation

  • The roate(angle) function can rotate "an element around the point of origin by a specified angle value."
  • The following are valid provided angle value types:
    • deg
    • grad
    • rad
    • turn

Skew

  • The skew(x, y) function "specifies a skew along the x and y axes."
  • 'x' specifies the skew on the x axis, and 'y' specifies the skew on the y axis; both are provided in deg, turn, rad, or grad.

Changing the Origin of the Transform

  • The transform-origin property defaults to the center of the object but can be changed.

Choose Your Ordering Carefully

  • Transform functions' order DOES matter: "if you rotate before translating, your translate direction will be on the rotated axis."
