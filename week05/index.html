Feature Detection

  • IF statements can detect features like if (window.holoDeck) and then activations to other features could be placed in the THEN block.
  • "Can I Use?" is a good resource for checking which features are available in different browsers and versions.
  • "browser sniffing" (not recommended) can also be used as the 'old-school' way for finding browser support features.
