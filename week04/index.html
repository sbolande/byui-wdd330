Forms

  • Forms contain elements of type: input, select, button, etc.
  • Forms are processed using back-end, but it's becoming more common to pre-process via the front-end.
  • Each form control starts with an initial value that is changed via user input or JavaScript.

Accessing Form Elements

  • Legacy DOM provides const form = document.forms[0]; to be able to access the list of forms on the document.
  • The const form = document.forms['form_name']; also works, and const input = form['input_name']; works for accessing form elements.

Form Properties and Methods

  • A form can be submitted by a submit input, submit button, or an image input. form.submit(); works from the JavaScript.
  • A form can be reset by a reset button or the JavaScript form.reset();.
  • The action property can be changed by using form.action = 'different_url';.
