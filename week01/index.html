Lesson 1, Step 1

  • Because of the diversity in screen sizes, there is a higher focus on Mobile-first development.
  • "We need to rewire our brains."
  • Starting small and scaling up shows focus on content and functionality, helps you scale upward, and helps the end user.

Lesson 1, Step 2

  • Steps to understand the user
    • Be There
    • Be Useful
    • Be Quick
  • Improve the user experience by cutting steps for the user
  • Improve the user experience by putting things in reachable places
  • Speed matters!!

Lesson 1, Step 3

  • Keep things simple, put things in an accessible place to increase user experience.
  • "Created perception of speed" = increased user satisfaction
  • Understanding, Familiarity, and Perception matters
  • "...we need to ensure that we evolve with the technology..."

Lesson 1, Step 4

  • Larger targets can decrease cognitive load
  • Features that reduce errors and increase ease of access like autocomplete can help the user
  • For number inputs and email inputs use non-default keyboards if possible
  • "Don't force a copy and paste", calls matter

Lesson 1, Step 5

  • "End-users...shape the market."
  • Research!!!
  • Make ads simple and effective
  • STOP USING POP-UPS/POP-OVERS
  • Keep ads unobtrusive
