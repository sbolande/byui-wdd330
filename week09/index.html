Going Global

  • Global browser environments are "stored" on the window object.
  • You should generally avoid referencing globals through the window object as it's more portable.

Dialogs

  • The alert(); confirm(); prompt(); confirm(); functions are not JS, they're part of the window obj.

Browser History

  • window.history contains the current session history.
  • The window.history.go(1); function can go forward one page in history.

The data- Attribute

  • You can embed data in a web page using custom attributes by preceding the attributes with data-. Each element has a dataset property can be used to access any data- attributes.
