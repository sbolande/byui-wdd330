A Brief History of Ajax

  • "Ajaj" shows the most commonly used AJAX returns JSON.
  • Ajax is used more than ever to transport data back and forth between sites.

Fetch

  • Fetch API uses promises to avoid callback hell.
  • The fetch() method returns a promise that resolves to the response returned from the URL that was provided as an argument, but should also include a catch statement to deal with any errors that may occur on promise rejection.
  • Any response between 200 and 299 is considered "ok" on the response object.
  • The redirect() method, while not supported in any browser, could be used to create a new promise that resolves to the response form the redirected url.
  • Text, File, and JSON responses can all come back on AJAX, but JSON is the most common format.
